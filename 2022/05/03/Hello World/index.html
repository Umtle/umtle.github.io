Hello World , Hello Hexo

发表于2022-05-03|更新于2022-05-04|个人项目
|阅读量:

最近因疫情原因学校封了宿舍，闲的要死，正巧和一位写博客的学长聊起来了，才想起自己还有个挂掉很久的博客。
闲着也是闲着，就顺手把博客迁移到了 github pages，顺便也玩玩 Hexo。
配置过程基本参考了CSDN的这篇文章和 Hexo官方文档，重新整理记录一些踩过的坑。

什么是 Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。
—— Hexo官方文档

使用npm安装Hexo

Node.js和Git安装完成后，使用npm来安装Hexo。

同样检测下是否安装成功，返回版本号即可。

在GitHub建立仓库

登陆后，点击右上角的加号创建新仓库（New repository），创建一个名为username.github.io的仓库，其中username需为用户id的小写。

进入仓库，在settings\pages选项卡中设置页面来源（Source）和自定义域名（Custom domain）。可以先使用默认域名username.github.io，测试完成之后再修改为自己的域名。
